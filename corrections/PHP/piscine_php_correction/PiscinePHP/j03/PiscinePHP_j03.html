Les préliminaires c'est important !

- Tout le monde est présent ?
- faites alors un git clone du repo dans votre vhost j03. Le but est que vos propres accès sur http://j03.local.42.fr:8080/ arrivent sur la racine du rendu
- Commencez la soutenance.

On utilise Chrome (celui du dump !) et curl (celui du dump !) pour cette soutenance.

Si il n'y a aucun rendu, mettez 0 et arrêtez l'évaluation.
Discutez de ce qui a posé problème (pas forcement au niveau technique) et de comment éviter la même situation demain.

ex00 vhost

Le but était d'installer MAMP et de configurer un vhost http://j03.local.42.fr:8080/

ex00 vhost


- dans le rendu bitnami-apps-vhosts.conf a bien été modifié pour inclure ~/mamp/apps/j03/conf/httpd-vhosts.conf ? 1pt

il faut que le path soit complet, par exemple: 


$> more bitnami-apps-vhosts.conf

# Bitnami applications installed in a Virtual Host

Include "/Users/xlogin/mamp/apps/j03/conf/httpd-vhosts.conf"


- dans le rendu bitnami-apps-vhosts.conf a bien été modifié pour definir le bon DocumentRoot et inclure ~/mamp/apps/j03/conf/httpd-app.conf ? 2pt 


$> more httpd-vhosts.conf 

<VirtualHost *:8080>

  ServerName j03.local.42.fr

  DocumentRoot "/Users/xlogin/mamp/apps/j03/htdocs"

  Include "/Users/xlogin/mamp/apps/j03/conf/httpd-app.conf"                                                                                           

</VirtualHost>


- dans le rendu ex00/httpd-app.conf a bien été modifié pour utiliser ~/mamp/apps/j03/htdocs ? 2pt 


$> more httpd-app.conf 

<Directory "/Users/xlogin/mamp/apps/j03/htdocs">

    Options Indexes MultiViews

    AllowOverride All

    <IfVersion < 2.3 >

    Order allow,deny

    Allow from all

    </IfVersion>

    <IfVersion >= 2.3>

    Require all granted

    </IfVersion>

</Directory>

Rate it from 0 (failed) through 5 (excellent)

ex03 cookie_crisp

Le but était de créer une page cookie_crisp.php qui permet de créer, lire, et effacer un cookie.

ex03 cookie_crisp

- créer un cookie, il faut que la page renvoie bien dans le header un set-cookie -> +2pt 


$> curl -v 'http://j03.local.42.fr:8080/ex03/cookie_crisp.php?action=set&name=boisson&value=white_russian'

...

< Set-Cookie: boisson=white_russian; expires=...

...


- relire un cookie créé --> +2pt 


$> curl -c cook.txt 'http://j03.local.42.fr:8080/ex03/cookie_crisp.php?action=set&name=dessert&value=tiramisu'

$> curl -b cook.txt 'http://j03.local.42.fr:8080/ex03/cookie_crisp.php?action=get&name=dessert'

tiramisu

$>


- effacer le cookie "dessert" que l'on vient de créer (uniquement si les 2 tests précédents sont bons évidemment) -> +1pt 


$> curl -c cook.txt 'http://j03.local.42.fr:8080/ex03/cookie_crisp.php?action=del&name=dessert'

$> curl -b cook.txt 'http://j03.local.42.fr:8080/ex03/cookie_crisp.php?action=get&name=dessert'

$>
